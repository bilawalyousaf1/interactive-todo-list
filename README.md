# Interactive Todo List 📝

A responsive and modern Todo List web application that allows users to add, edit, and delete tasks with smooth animations for enhanced UI.

## 🚀 Features
- Add, edit, and delete tasks dynamically
- Smooth animations for header, tasks, and container
- overlay gradient
- Interactive buttons with hover effects
- Responsive layout for mobile and desktop
- Glassmorphism effect for UI polish

## 🛠️ Built With
- HTML5 (Semantic markup, form handling)
- CSS3 (Flexbox, Grid, animations, gradients, responsive design)
- JavaScript (DOM manipulation, event handling, dynamic updates)
- Font Awesome icons
- Google Fonts (Poppins)

## 📂 Project Structure
- index.html → Page structure and DOM elements
- style.css → Styling, animations, responsive design
- main.js → Logic and interactive functionality
- assets/ → Images, video, icons (if any)

## 📚 What I Learned
- HTML: Semantic structure, forms, external resources
- CSS: Flexbox, Grid, keyframe animations, gradients, transitions, responsive design
- JavaScript: DOM manipulation, crud operation, event handling, functions, template literals, conditional logic, parent-child relationships
- UI/UX: Modern, interactive, user-friendly design
- Responsive & interactive interfaces across devices

## ▶️ How to Run
Open `index.html` in a web browser. Use the form to add tasks, edit with prompts, and delete tasks dynamically.
